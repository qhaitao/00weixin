# 领导讲话 / 发言稿写作方法论

本文档是 `doc-writer` 处理 `speech` 类型文档的写作参考，供 Agent 撰写时调用。

---

## 一、标题设计

领导讲话的标题决定场合定位和文档属性，四种类型按场景选用：

| 类型 | 格式 | 适用场景 | 示例 |
| --- | --- | --- | --- |
| **场合型**（最常用） | 在[会议名称]上的讲话 | 有明确会议场合的正式讲话 | 在2026年全省财务工作会议上的讲话 |
| **主题型** | 关于[核心主题]的几点意见 | 无固定会议、专题性发言 | 关于加强财务内控建设的几点意见 |
| **任务型** | 在[场合]上关于[主题]的讲话 | 场合和主题各占重要性 | 在年度部署会上关于降本增效的讲话 |
| **述职型** | [时间段]工作述职报告 | 个人工作回顾汇报 | 2026年上半年工作述职报告 |

**判断流程**：
1. 有明确会议名称 → 场合型
2. 有核心政策/工作主题，无固定会议 → 主题型
3. 两者皆有 → 任务型
4. 个人回顾汇报 → 述职型

**禁止使用**的标题模式：
- "关于某某问题的探讨"（学术味，不是党政语言）
- "我的发言"（过于随意，缺乏正式感）
- 标题超过20字（冗长，记忆成本高）

## 二、金字塔原理在讲话中的应用

### 核心原则：结论先行（Top-Down）

每一个层级都从结论开始，支撑材料跟在后面。听众在第一句就知道这节讲什么。

### 三层结构模型

```
第①层：结论句（金句）—— 节首，15-20字，是整节最有分量的一句话
    │
    ├── 第②层：支撑 —— 1-2句，数据/事实/对比，说明"为什么"
    │
    └── 第③层：举措 —— 三条可执行行动，"一要…；二要…；三要…"
```

**检验标准**：把第①层结论句单独拿出来，这句话有没有独立的传播价值？如果没有，需要重写。

### 全篇结构类型

#### 并列式（最常用）
适用于三项独立工作任务，各节同等重要：
```
开场白
一、[论点A]  ——  结论①→支撑②→举措③
二、[论点B]  ——  结论①→支撑②→举措③
三、[论点C]  ——  结论①→支撑②→举措③
结语（金句收尾）
```

#### SCQA 式（适合研判+部署类）
先讲形势挑战，设问，再给出答案和部署：
```
开场白
一、总体形势（SCQA导入）
    S - 当前形势（1句话定基调，正面）
    C - 面临挑战（数据支撑，引发紧迫感）
    Q - 核心问题（设问，引出答案）
    A - 总体思路（亮明总论点，即后续各节的浓缩版）
二、[论点A：第一项部署]  ——  结论①→支撑②→举措③
三、[论点B：第二项部署]  ——  结论①→支撑②→举措③
四、[论点C：第三项部署]  ——  结论①→支撑②→举措③
结语（金句收尾）
```

#### 递进式（适合从认识到行动的讲话）
三节层层推进，有因果递进关系：
```
开场白
一、[提高认识：Why]   ——  为什么重要
二、[明确部署：How]   ——  怎么做
三、[展望目标：Where] ——  达到什么
结语
```

## 三、金句写作公式

金句是讲话中最容易被记住的句子，通常出现在三个位置：
- **开场**：引用型，定基调（引习近平讲话、政策文件权威表述）
- **各节首句**：核心论点型，有力表达该节结论
- **结语**：号召型/排比型，全篇最高潮

### 五种公式

| 类型 | 公式结构 | 示例 | 适用位置 |
| --- | --- | --- | --- |
| **排比型**（最常用） | 要A、要B、更要C | 要敢想、敢干、更要敢担当 | 节首句、结语 |
| **对仗型** | 既要A，又要B | 既要防范风险，又要激发活力 | 节首句 |
| **数据型** | 从A到B，X年实现C | 三年间，利润从亏损到盈利两亿 | 节首句、SCQA支撑 |
| **设问型** | A靠什么？靠B | 转型靠什么？靠改革、靠创新、靠实干 | 节首句、SCQA问题 |
| **号召型** | 让我们A、B，共同C | 让我们齐心协力、攻坚克难，共同书写新篇章 | 结语 |

### 金句打磨四步法

1. **提取素材**：从事实中找到有对比、有数据、有三要素的原始材料
2. **套用公式**：选择最匹配的金句类型套用
3. **压缩字数**：反复删减到15-20字，删掉所有可以不说的词
4. **朗读检验**：读出来，停顿自然、节奏感强才算过关

## 四、语气与措辞

| tone | 适用场合 | 高频词 | 避免 |
| --- | --- | --- | --- |
| **confident** | 动员部署、表态承诺 | 坚决、切实、全力以赴、务必、不折不扣 | 过于强硬、官僚腔 |
| **neutral** | 工作汇报、情况说明 | 积极推进、扎实落实、持续深化、下一步将 | 太平淡、无力度 |
| **humble** | 述职检查、问题剖析 | 有差距、不足、努力方向、反思、改进 | 过于自责、失去信心感 |

## 五、常见错误

| 错误 | 症状 | 修正方向 |
| --- | --- | --- |
| **节首无金句** | 第一句话是背景铺垫，不是结论 | 把结论移到节首，背景放到②支撑 |
| **举措不可执行** | "要高度重视""要切实加强" | 改为"要在X月前完成Y""要建立Z机制" |
| **支撑无数据** | "工作成效显著，业务快速发展" | 加具体数字、对比、时间跨度 |
| **结语无金句** | "以上就是我的发言，谢谢大家" | 结语须有一句排比/号召型金句，然后再"谢谢" |
| **SCQA的A太弱** | A部分只有一句泛化口号 | A须是后续各节的浓缩版，读者读完A能预判后续内容 |
| **标题缺年份** | "在财务工作会议上的讲话" | 加年份："在2026年广东邮政财务工作会议上的讲话" |
