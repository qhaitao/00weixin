# ppt-outline — PPT 大纲生成器

## 职责

读取 `analysis.json`，生成结构化的 PPT 演讲大纲（`ppt-outline.md`），遵循咨询级幻灯片设计原则：

- **结论先行**：每页标题均为完整句子结论（Action Title），不是标签
- **金字塔结构**：论点 MECE 分布，每页只讲一个观点
- **SCQA 叙事**：为 persuasive 型演示提供情境-冲突-问题-答案框架
- **可视化建议**：每页指定图表类型、数据来源和讲解要点

## 前置条件

- `analysis.json` 已由 Agent 填写完整（通过 `content-analyzer` → Agent 分析）
- 可选：`draft.md` 已由 Agent 补全（大纲会引用其中内容）

## 调用方式

```powershell
bun skills\ppt-outline\scripts\main.ts --dir workspace\任务目录
```

## 输出

| 文件                    | 说明                                                                  |
| ----------------------- | --------------------------------------------------------------------- |
| `ppt-outline.md`      | 幻灯片大纲（每页含 action title、类型、要点、可视化建议、讲解词引导） |
| `_ppt_instruction.md` | 给 Agent 的定制指令（可进一步调整风格或结构）                         |

## 输出格式示例

```
# PPT 大纲：2026年Q2财务分析报告

## Slide 1 | 封面
- **类型**：title-slide
- **标题**：2026年第二季度财务分析报告
- **副标题**：中国邮政集团有限公司广东省分公司 | 2026年4月
...

## Slide 2 | 执行摘要
- **行动标题**：Q2 营收同比增长 8.5%，利润增长 12.3%，三大指标全线达标
- **类型**：content
- **核心信息**：...
- **要点**（≤4条）：
  1. ...
- **可视化建议**：关键指标仪表盘
- **讲解词引导**：...
```

## 设计原则

1. **每个标题都是完整句子**（不是标签），单独读标题就能理解全篇逻辑
2. **每页只讲一个观点**，≤4条要点，每条 10-15 字
3. **可视化优于文字**：优先用图表/流程图/对比表替代段落文字
4. **讲解词补充而非重复**：Speaker Notes 增加叙事，不照念 slide 文字
5. **页数控制**：报告型 20-30 页，发言稿30-50页，纪要 5-8 页，通知 3-5 页

## 注意

- 本 skill 不生成实际 PPTX 文件。如需 PPTX 输出，可将 `ppt-outline.md` 交给 Agent 使用 PptxGenJS 或 Marp 渲染。
- 大纲生成后须人工审核标题是否准确、逻辑是否通顺、数据是否最新。
